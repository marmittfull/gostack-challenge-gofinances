This challenge aims to build an application that receives .csv files and lists to the user all the monetary transactions saved in the document, also calculating when it was deposited, withdrawn and the total.

## üî• Functionalities:

Through this application we can:

- Import .csv files
- View all transactions on a friendly interface
- Look at the received and spent money (with the total appended next to it)
